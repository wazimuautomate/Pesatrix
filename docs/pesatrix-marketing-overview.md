# Pesatrix Marketing Overview

## What Pesatrix Is

Pesatrix is a Kenya-first online earning platform that helps users make money from real digital tasks, learn how to submit quality work, track their earnings transparently, and withdraw cleared funds directly to M-Pesa.

It is built for people who want a practical, guided way to earn from online work without guessing which sites are real, how tasks are reviewed, or when earnings become available. Pesatrix brings the full earning journey into one platform: account creation, onboarding, activation, training, task access, review, wallet tracking, referrals, support, and M-Pesa withdrawals.

Pesatrix is not positioned as a get-rich-quick scheme or a fixed-income promise. Earnings depend on available tasks, user eligibility, task quality, review decisions, account standing, and current platform rules. That honesty is part of what makes Pesatrix different. Instead of promising instant wealth, Pesatrix gives users a structured system for building trust, learning task standards, completing approved work, and receiving eligible payouts through familiar Kenyan payment rails.

## How Pesatrix Works

### 1. Create An Account

A user starts by registering with their full name, Kenyan phone number, county, email address, password, and optional referral code.

The phone number matters because it becomes part of the user identity and payout flow. Pesatrix validates Kenyan phone number formats and uses account details to support activation, withdrawals, referrals, support, and fraud protection.

### 2. Confirm Email And Sign In

After registration, the user confirms their email and signs in using their account credentials. This protects the account and keeps important platform activity tied to the correct user.

### 3. Complete Onboarding

Before activation, Pesatrix walks the user through the main dashboard sections:

- Dashboard
- Tasks
- Rewards
- Profile
- Training
- Wallet
- Support
- Referrals

This helps users understand the earning sequence before they continue. Pesatrix makes the flow clear: onboarding first, activation next, training after that, then live task work after compliance checks.

### 4. Activate The Account With M-Pesa

Pesatrix uses a one-time M-Pesa activation flow. The product currently reads the activation amount from platform settings and initiates activation through M-Pesa STK Push.

Activation unlocks the earning flow, but it does not guarantee fixed earnings. It prepares the account for training, tasks, wallet activity, referrals, and withdrawals.

The activation system is built with real payment tracking:

- STK Push initiation
- Pending activation records
- M-Pesa callback validation
- Receipt checks
- Amount matching
- Phone matching
- Duplicate receipt protection
- Account activation sync after confirmed payment

This makes the activation process more serious and traceable than a manual payment promise.

### 5. Complete The 7-Day Training Program

Pesatrix does not simply open tasks to every new account. Users complete a guided training path first.

The training program teaches users how online work is reviewed and why quality matters. It covers:

- Participant integrity
- Honest survey participation
- Screeners and attention checks
- Technical setup for online tasks
- Data labeling basics
- Ground truth and labeling accuracy
- Transcription and production skills
- Spatial annotation and computer vision tasks
- NLP annotation and evaluation
- QA testing discipline
- AI response evaluation, RLHF, red teaming, and human-in-the-loop workflows

Training is organized into three stages:

| Stage | Level | Focus |
| --- | --- | --- |
| Foundation | Beginner | Integrity, survey quality, profile consistency, and data labeling basics |
| Production Skills | Intermediate | Transcription, annotation, spatial labeling, and reviewer-grade accuracy |
| Expert Evaluator | Advanced | NLP annotation, QA testing, AI evaluation, hallucination checks, and red teaming |

Users must pass quizzes and stage checks. This protects the platform from poor submissions and helps serious users understand how to earn correctly.

### 6. Unlock Tasks

After onboarding, activation, and training, task access can unlock. Pesatrix can also apply a task preparation window after training completion. A successful activated referral can reduce the remaining wait time, depending on current platform settings.

This makes task access more controlled and more reliable. Users do not just arrive and spam submissions. They build account readiness first.

### 7. Browse Available Tasks

The Tasks area shows live work with clear details:

- Task title
- Category
- Description
- Instructions
- Payout in KSh
- Difficulty
- Remaining slots
- Expiry status
- Proof requirements
- Minimum word count where relevant
- Whether screenshots or submitted links are required
- Whether the user can start or only preview the task

Users can filter and sort tasks by category, difficulty, payout, expiry, and search terms.

### 8. Complete Different Types Of Digital Work

Pesatrix supports several task categories:

| Category | What Users Do |
| --- | --- |
| Surveys | Answer structured questions such as multiple choice, yes/no, ratings, multi-select, and open text |
| Data Labeling | Label batches of text or image items using defined label options |
| Social Engagement | Follow, subscribe, like, comment, share, join, review, rate, download, or purchase, then submit proof |
| Verification | Submit text answers, screenshots, URLs, or mixed proof depending on the task |
| Content Creation | Write short text, paragraphs, articles, tweets, captions, reviews, or other original content |
| Watch & Respond | Watch content for a required period and answer comprehension questions |
| Partner Surveys | Access partner survey opportunities when configured, including CPX Research integration |

This variety matters because not every user is good at the same kind of work. Pesatrix can support simple tasks, writing tasks, proof-based tasks, AI-data tasks, and partner research tasks from one dashboard.

### 9. Submit Proof And Answers

Each task has its own submission rules. Some require written answers. Some require screenshot proof. Some require a submitted URL. Social tasks can require a username, exact comment text, and uploaded image proof. Watch tasks require a valid watch session and can pause or forfeit progress if the user repeatedly leaves the page.

Pesatrix is strict because approved work is connected to wallet credit. The platform checks for:

- Missing required answers
- Missing screenshots
- Invalid submitted links
- Duplicate submissions
- Too-fast completion
- Duplicate answer patterns
- Uniform survey answers
- Duplicate screenshots
- Social proof quality
- Watch-time compliance
- Account risk indicators

This protects the value of the work and rewards users who follow instructions.

### 10. Review And Approval

Submitted work can move through statuses such as pending, AI reviewing, approved, declined, flagged, or admin reviewed.

Pesatrix uses a mix of automated and human review. AI grading can review task answers, screenshots, content quality, data labeling accuracy, verification proof, and watch-and-respond answers. When AI is uncertain or unavailable, submissions can be flagged for manual review.

Approved work credits the wallet. Declined or incomplete work does not qualify. Flagged work receives additional review.

### 11. Track Earnings In The Wallet

The Pesatrix wallet separates money into clear states:

- Available balance: earnings that can be considered for withdrawal.
- Pending balance: earnings still in a hold or review period.
- Total earned: lifetime earnings recorded by the platform.
- Transaction history: money in, money out, task earnings, referral bonuses, rewards, deposits, withdrawals, reversals, and related records.

This is one of the strongest parts of Pesatrix. Users do not have to wonder where their money is. The wallet shows what has cleared, what is pending, and what has already been withdrawn.

### 12. Withdraw To M-Pesa

Pesatrix supports withdrawal requests to the user account's M-Pesa number.

The withdrawal flow includes:

- Available-balance checks
- Minimum withdrawal checks
- Processing fee display
- Amount-to-receive calculation
- Kenyan M-Pesa phone validation
- Account-phone matching
- One-active-withdrawal protection
- Daily withdrawal amount limits
- Daily withdrawal count limits
- Single-request limits
- Risk checks before payout
- Admin payout queue
- M-Pesa B2C payout processing
- Failure handling and ledger reversal

Users see the amount requested, processing fee, amount to receive, and estimated processing timeline before completing the request.

### 13. Earn From Direct Referrals

Pesatrix gives users a personal referral code and referral link. When someone signs up through the link and completes paid activation, the direct referrer can earn a referral bonus according to the current product rules.

The referral dashboard shows:

- Referral link
- Referral code
- Available referral earnings
- Pending referral earnings
- Direct referral count
- Bonus amount
- Referral pipeline
- Latest referral bonuses

Referral bonuses are guarded by fraud checks. Pesatrix can block suspicious referral rewards when accounts appear to share device or IP patterns.

### 14. Use Dashboard Support

Pesatrix includes in-dashboard support tickets for account, activation, training, tasks, wallet, withdrawal, referral, and safety issues.

This matters because support requests stay tied to the user's account and records. The support team can review the issue with the right context instead of asking the user to repeat details already stored in the platform.

### 15. Grow Into Future Rewards

The dashboard includes a Daily Rewards area currently presented as "coming soon." Planned reward experiences include:

- Spin and win
- Daily check-in bonuses
- Referral streak rewards
- Milestone drops

The product also contains reward-wheel logic for activated accounts, including free and paid spin models, wallet credits, daily limits, and streak tracking. The visible dashboard currently presents this area as upcoming, so marketing should describe Daily Rewards as a feature being built or rolling out, not as the main active earning method.
